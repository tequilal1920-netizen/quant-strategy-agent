# 行业景气度

## 目标

把行业基本面景气、市场风格和配置策略整合为一套时点一致、可回放且可审计的行业研究流程。

## 输入

- 研究截止日、申万行业口径、行业景气指标和市场风格指标。
- 事件覆盖、可投资行业、基准、持仓、成本与换手约束。
- 可选的策略选择、历史窗口和风险预算。

## 输出

- 行业景气度排名、证据分解和数据质量说明。
- 风格轮动状态、主要驱动与历史相似阶段。
- 一个或多个配置策略的行业权重、风险、换手和回测结果。
- 供主页和组合优化消费的标准行业信号。

## 工作流

1. 阅读 `references/module-map.md`、模型 README、事件覆盖规则和当前状态文件。
2. 核对行业分类、截止日、财报可用时点、价格时间点和事件发生时间。
3. 区分原始景气、风格信号和人工事件覆盖；保留每项覆盖的理由与有效期。
4. 运行正式快照和跟踪入口，不在页面端重新计算核心模型。
5. 检查行业排序、风格状态、策略权重、跟踪误差、换手和成本。
6. 遍历三个二级页面及其页内控件，确认旧图表均已映射且没有重复点击处理器。

## 约束

- 不使用未来财报、未来成分股或事后事件信息。
- 行业景气、风格轮动和配置策略必须保持独立字段，不能互相冒充。
- 状态颜色固定为绿色正常、蓝色运行、红色异常。
- 中文楷体、英文和数字 Arial；遵守统一看板布局。
