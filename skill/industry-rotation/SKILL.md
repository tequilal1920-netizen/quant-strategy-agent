# 行业景气度

## 目标

把31行业专属业务景气、季度股票级3×4风格箱和配置策略整合为一套时点一致、可回放且可审计的研究流程。

## 输入

- 研究截止日、申万行业口径、31×8专属业务景气指标和全体合格A股底表。
- 季度流通市值、PIT财务可见日、复权价格、估值与股息率字段。
- 事件覆盖、可投资行业、基准、持仓、成本与换手约束。
- 可选的策略选择、历史窗口和风险预算。

## 输出

- 行业景气度排名、证据分解和数据质量说明。
- 逐股大/中/小盘与成长/均衡/价值/红利标签、12箱覆盖、迁移和Top3信号。
- 一个或多个配置策略的行业权重、风险、换手和回测结果。
- 供主页和组合优化消费的标准行业信号。

## 工作流

1. 阅读 `references/module-map.md`、模型 README、事件覆盖规则和当前状态文件。
2. 核对31行业、每行业8项互异专属字段、截止日、价格时间点和事件发生时间。
3. 先运行行业快照，再运行 `style_box_rotation.py`；风格财务必须满足 `visible_date <= signal_date`。
4. 检查合格股票数=已标记数=唯一股票数、未分类=0、重复=0、12格无空箱。
5. 检查行业月/周Top10各10%与同频31行业等权基准；检查风格季度Top3各1/3与12箱等权基准。
6. 核对训练/验证/测试分段，候选只由训练与验证选择；测试集不得反向调参。
7. 运行正式跟踪入口，不在页面端重新计算核心模型。
8. 检查排序、归因、策略权重、跟踪误差、换手、成本和发布门。
9. 遍历三个二级页面及其页内控件，确认旧图表均已映射且没有重复点击处理器。

## 约束

- 不使用未来财报、未来成分股或事后事件信息。
- 行业字段不得使用统一收入利润、ROE/ROA、毛利率负债率、股价动量趋势、换手拥挤或估值凑数。
- 风格频率固定季度；规模与风格标签对每只合格股票必须互斥且穷尽。
- 行业景气、风格轮动和配置策略必须保持独立字段，不能互相冒充。
- 状态颜色固定为绿色正常、蓝色运行、红色异常。
- 中文楷体、英文和数字 Arial；遵守统一看板布局。
