# 资产配置

## 目标

生成训练、验证、测试隔离且可复现的大类资产配置结果，并向主页提供日度、周度和月度配置建议。

## 输入

- 宏观周期、资产收益与风险数据的截止日和频率。
- 可投资资产集合、基准、风险预算、成本和换手约束。
- 策略选择与可选参数覆盖。

## 输出

- 周期跟踪状态、证据和历史位置。
- 各配置策略的目标权重、风险贡献、预期换手和回测摘要。
- 供主页和组合优化消费的标准快照。

## 工作流

1. 阅读 `references/module-map.md`、模型 README 和当前状态文件。
2. 核对资产、时间点、币种、频率、基准和成本口径。
3. 保持训练集筛选、验证集定型、测试集仅报告；禁止用测试集调参。
4. 运行唯一正式入口并更新正式快照，不创建平行版本。
5. 检查权重和为 1、资本权重、风险贡献、换手、成本与压力情景。
6. 验证“周期跟踪”和“配置策略”的全部页内控件、图表和结论卡。

## 约束

- 不允许未来数据、测试集选参或无说明的主观覆盖。
- 状态颜色固定为绿色正常、蓝色运行、红色异常。
- 中文楷体、英文和数字 Arial；遵守统一看板布局。
- 结果写入 `output/`，数据库留在 `database/`。
