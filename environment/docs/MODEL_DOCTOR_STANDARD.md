# Model Doctor Standard

本文件定义模型效果诊断的最低深度。

## 禁止的诊断方式

- 只看年化收益。
- 只看 Sharpe。
- 收益不好就调参数。
- 回撤大就直接降仓位。
- 看到某个因子差就删除。
- 为了通过测试集反复试参数。

## 必须的穿透路径

### 1. 从组合结果往上穿透

```text
总收益
  -> 资产/行业/个股贡献
  -> 权重变化
  -> 信号分数
  -> 原始特征
  -> 数据源和处理规则
```

### 2. 从风险结果往上穿透

```text
最大回撤
  -> 回撤期间持仓
  -> 暴露变化
  -> 信号是否失效
  -> 交易是否滞后
  -> 风控是否缺失
```

### 3. 从交易成本往上穿透

```text
成本拖累
  -> 换手
  -> 调仓频率
  -> 信号稳定性
  -> 组合约束
```

### 4. 从 Alpha 衰减往上穿透

```text
IC 下降
  -> 因子分层收益
  -> 因子拥挤
  -> 行业/风格暴露
  -> 数据覆盖变化
  -> 市场 regime 切换
```

## 实验建议的证据要求

每个迭代候选必须写：

- 诊断问题。
- 证据指标。
- 对应报告或方法论依据。
- 修改等级。
- 预期改善路径。
- 风险。
- 回滚条件。

## 验收矩阵

至少同时检查：

- 年化收益。
- Sharpe。
- 最大回撤。
- Calmar。
- 换手。
- 成本。
- 样本内/外。
- 年度切片。
- 参数敏感性。
- 归因变化。
